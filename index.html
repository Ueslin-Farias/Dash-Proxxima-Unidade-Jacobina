✅ Esquema Estruturado (Google Sheets) — Detectado Automaticamente

Quando a planilha contém a coluna Tipo, o sistema usa o esquema estruturado com as regras de negócio abaixo. Se não houver coluna Tipo, usa o esquema legado do arquivo XLSX.

✅ Conta como VENDA (fatura)
Tipo = Venda
Status = APROVADO
Status = EM APROVAÇÃO
❌ Conta como PERDA (não fatura)
Tipo = Perda
Status = CANCELADO
Status = REJEITADO
🔁 Deduplicação automática por ID — registros com ID duplicado são ignorados automaticamente para evitar contagem dupla.

🌐 Publicar Dashboard Online (Google Sites / GitHub)
📡 Como deixar o dashboard acessível para toda a equipe

O dashboard é um arquivo index.html único — para publicá-lo sem servidor:

🐙 GitHub Pages (recomendado, grátis)
  1. Crie uma conta em github.com
  2. Crie um repositório público (ex: dashboard-proxima)
  3. Faça upload do index.html
  4. Vá em Settings → Pages → Source: main / root
  5. URL gerada: seuuser.github.io/dashboard-proxima
  6. Para atualizar: substitua o index.html e aguarde ~1min
📋 Google Sites com iframe
  1. Publique o HTML no GitHub Pages (passo ao lado)
  2. No Google Sites, adicione bloco "Incorporar"
  3. Cole a URL do GitHub Pages
  4. O dashboard aparece embutido no site
  5. Quando você atualizar o index.html no GitHub, o Google Sites reflete automaticamente
🔄 Fluxo para atualizar os dados sem que a equipe precise fazer nada

1. Você importa o .xlsx novo aqui no dashboard → 2. Clica em "☁️ Exportar p/ Nuvem" → 3. Cola o JSON num GitHub Gist público → 4. Salva a URL raw nas configurações acima → 5. Qualquer pessoa que abrir o link verá os dados atualizados automaticamente (Auto-Load).

🔗 Google Sheets — Sincronização Automática
📊 Como conectar ao Google Sheets
📋 Estrutura da planilha (colunas obrigatórias)
ID  ·  Nome Colaborador
Data Venda (YYYY-MM-DD)
Semana (S1, S2, S3, S4)
Mês  ·  Valor Venda
Status  ·  Tipo (Venda/Perda)
Tipo=Venda → fatura · Tipo=Perda → contabiliza perda
Status "EM APROVAÇÃO" → conta como venda
🚀 Como publicar como CSV
  1. Abra a planilha no Google Sheets
  2. Vá em Arquivo → Publicar na web
  3. Selecione a aba com os dados
  4. Escolha formato Valores separados por vírgulas (.csv)
  5. Clique Publicar e copie a URL gerada
  6. Cole abaixo e clique em Salvar
URL da planilha publicada como CSV
—
💡 O dashboard sincroniza automaticamente ao carregar se a URL estiver configurada. Também é possível usar arquivo .xlsx/.csv local como alternativa.
🔄 Compartilhar Dashboard (Backup Local)

Use Exportar Backup para gerar um arquivo .json com TODOS os dados (metas, fotos, status, importações, semanas, histórico). Quem receber o arquivo pode usar Importar Backup para ver exatamente o mesmo dashboard que você — ideal para compartilhar via WhatsApp/Drive.
