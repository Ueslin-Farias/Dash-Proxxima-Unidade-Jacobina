🌐 Publicar Dashboard Online (Google Sites / GitHub)
📡 Como deixar o dashboard acessível para toda a equipe

O dashboard é um arquivo index.html único — para publicá-lo sem servidor:

🐙 GitHub Pages (recomendado, grátis)
  1. Crie uma conta em github.com
  2. Crie um repositório público (ex: dashboard-proxima)
  3. Faça upload do index.html
  4. Vá em Settings → Pages → Source: main / root
  5. URL gerada: seuuser.github.io/dashboard-proxima
  6. Para atualizar: substitua o index.html e aguarde ~1min
📋 Google Sites com iframe
  1. Publique o HTML no GitHub Pages (passo ao lado)
  2. No Google Sites, adicione bloco "Incorporar"
  3. Cole a URL do GitHub Pages
  4. O dashboard aparece embutido no site
  5. Quando você atualizar o index.html no GitHub, o Google Sites reflete automaticamente
🔄 Fluxo para atualizar os dados sem que a equipe precise fazer nada

1. Você importa o .xlsx novo aqui no dashboard → 2. Clica em "☁️ Exportar p/ Nuvem" → 3. Cola o JSON num GitHub Gist público → 4. Salva a URL raw nas configurações acima → 5. Qualquer pessoa que abrir o link verá os dados atualizados automaticamente (Auto-Load).

☁️ Sincronização Online (Link Remoto)

Para que qualquer pessoa com o link do dashboard veja seus dados atualizados sem precisar importar arquivos:
1) Clique em Exportar p/ Nuvem e copie o JSON gerado. 2) Hospede em um GitHub Gist (raw URL), Dropbox (link direto) ou similar. 3) Cole a URL pública abaixo e clique em Salvar URL. Pronto — todos verão os dados ao abrir o dashboard.
